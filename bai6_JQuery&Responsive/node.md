- Responsive thể hiện sự phù hợp của website với các cỡ màn hình khác nhau
- Các kiểu responsive
    + Kiểu cổ điển: ứng với mỗi cỡ màn hình sẽ viết 1 website, sau đó dùng code để detech url nhằm phát hiện cỡ màn hình, sau đó điều hướng url tới website
    + Kiểu tân tiến: sử dụng htm 5 & css 3
- Một số khái niệm về responsive:
    + Responsive thực chất là việc viết lại css tương ứng với các cỡ màn hình khác nhau
    + @Media screen: Các thiết bị là màn hình
    + @Media all: tất cả các thiết bị
    + max-width: độ rộng tối đa
    + min-width: độ rộng tối thiểu
    + max-height: độ cao tối đa
    + min-height: độ cao tối thiểu
