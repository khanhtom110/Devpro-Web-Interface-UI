## 📚 TIMELINE CHƯƠNG TRÌNH HỌC (PHẦN FRONTEND)

Nội dung học được chia thành các buổi/module sau, tập trung vào việc áp dụng để hoàn thành giao diện:

| Buổi/Module | Keywords Đào tạo | Nội dung Chính |
| :--- | :--- | :--- |
| **1.** | **HTML** | Cấu trúc cơ bản, các thẻ Semantic HTML5, làm quen với VS Code. |
| **2.** | **CSS** | Định dạng cơ bản, Selector, Box Model, Flexbox, Animation & Transform (Tạo hiệu ứng). |
| **3.** | **Cắt Giao Diện từ Figma / PSD** | Thực hành chuyển đổi file thiết kế (Figma/PSD) thành code HTML/CSS. |
| **4.** | **JavaScript (JS)** | Cú pháp cơ bản, Biến, Hàm, Vòng lặp. Thao tác cơ bản với DOM (Document Object Model). |
| **5.** | **JQuery & AJAX** | Thư viện JQuery: Rút gọn thao tác DOM, xử lý sự kiện, tạo hiệu ứng nâng cao. Luyện tập **AJAX** (chuẩn bị cho việc tương tác Backend). |
| **6. & 7.** | **Responsive & Bootstrap 4** | **Responsive Design:** Sử dụng Media Queries để tối ưu hiển thị trên Mobile và Tablet. **Bootstrap 4:** Áp dụng Framework CSS phổ biến để xây dựng giao diện nhanh và chuẩn responsive. |
